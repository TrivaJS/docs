# Community Extensions

Browse extensions built by the Triva community.

## Popular Extensions

### Authentication & Security

- **triva-oauth** - OAuth 2.0 authentication
- **triva-passport** - Passport.js integration
- **triva-rbac** - Role-based access control

### Data & Validation

- **triva-joi** - Joi validation integration
- **triva-yup** - Yup validation integration
- **triva-sanitize** - Input sanitization

### Integrations

- **triva-stripe** - Stripe payment integration
- **triva-sendgrid** - SendGrid email integration
- **triva-aws** - AWS services integration
- **triva-firebase** - Firebase integration

### Utilities

- **triva-cache-manager** - Advanced caching
- **triva-queue** - Job queue support
- **triva-scheduler** - Task scheduling
- **triva-websocket** - WebSocket support

### API & Documentation

- **triva-swagger** - Swagger/OpenAPI docs
- **triva-graphql** - GraphQL server
- **triva-rest** - REST API utilities

## Submit Your Extension

Have an extension to share?

1. Publish to npm
2. Fork [trivajs/website](https://github.com/trivajs/website)
3. Add your extension to `/extensions`
4. Submit pull request
