# Official Extensions

Extensions maintained by the Triva core team.

## Available Extensions

### Middleware
- **[@triva/cors](cors.md)** - CORS middleware
- **[@triva/helmet](helmet.md)** - Security headers
- **[@triva/compression](compression.md)** - Response compression
- **[@triva/rate-limit](rate-limit.md)** - Advanced rate limiting

### Authentication
- **[@triva/jwt](jwt.md)** - JWT authentication
- **[@triva/session](session.md)** - Session management

### Utilities
- **[@triva/multer](multer.md)** - File upload handling
- **[@triva/validator](validator.md)** - Request validation

---

## Installation

All official extensions are available on npm:

```bash
npm install @triva/extension-name
```
