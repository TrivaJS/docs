# Extensions

Extend Triva's functionality with official and community extensions.

## Overview

Extensions allow you to add new features to Triva without modifying the core framework.

### Official Extensions

- **@triva/cors** - CORS middleware
- **@triva/helmet** - Security headers
- **@triva/compression** - Response compression
- **@triva/jwt** - JWT authentication
- **@triva/rate-limit** - Advanced rate limiting
- **@triva/session** - Session management
- **@triva/multer** - File uploads
- **@triva/validator** - Request validation
