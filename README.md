This is the repository for the website [docs.trivajs.com](https://docs.trivajs.com/getting-started)

## Contributing

Contributing to Triva, in any of it's endevors is highly encouraged. Whether you are getting started with documentation or contributing, or you want to get down in the bunker and help us solve library problems, or build new library features, please review our [Contributing Guide](https://docs.trivajs.com/contributing) before making any efforts to commit changes.

## Building the Website locally

In most cases, changes will be content related, and will not require the cloning of this repo in order to make changes. But, in the case you'll need to download, please use the following steps:

1. Install Git
2. Clone this repo
```bash
$ git clone https://github.com/TrivaJS/docs.git
```
